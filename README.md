# Proxi

From problem to solution.

Proxi is a location-based service discovery platform that connects users
with nearby service providers such as mechanics, electricians, plumbers,
and technicians.

## Features
- Find nearby service providers
- Interactive map (Leaflet + OpenStreetMap)
- Provider registration
- Search & filters (service, distance, rating)
- Favorites / bookmarks
- Ratings & reviews
- Booking & scheduling
- Booking reminders (notifications)

## Tech Stack
- HTML
- CSS
- JavaScript
- Leaflet.js
- LocalStorage (MVP)
- OpenStreetMap

## Future Improvements
